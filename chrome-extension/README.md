# 使用 yter 下载（Chrome 扩展）

右键点击网页中的链接，在“使用yter下载”二级菜单中选择“下载视频”或“下载音频”，扩展会调用 yter 的 `/api/add_task` 接口创建任务。

## 安装

1. 打开 `chrome://extensions/`。
2. 开启右上角“开发者模式”。
3. 点击“加载已解压的扩展程序”，选择本目录 `chrome-extension/`。
4. 安装后会自动打开设置页。确认 yter 服务地址并点击“保存设置”。

默认服务地址是 `https://yter.cellmean.com`。如需使用其他部署地址，可点击浏览器工具栏中的扩展图标重新配置；保存自定义地址时，Chrome 会请求该站点的访问权限。

## 使用与验证

1. 在任意网页中右键点击一个链接。
2. 打开“使用yter下载”。
3. 选择“下载视频”或“下载音频”。
4. Chrome 通知显示任务编号即代表 yter 已接受任务；失败通知会显示连接或接口错误。

扩展只读取用户右键点击的链接，不注入页面脚本。菜单旁的图标由 Chrome 使用扩展的 16×16 图标显示；Chrome 的 `contextMenus` API 不支持为每个子菜单单独指定图标。
